*for first time or new user login only

2. git config --global user.email "email"
3. git Status (Opational for Status Check)

*for New Project/folder
1. git init (git ko initialize kr dega project floder ma)
2. (git add .) for compelet folder upload / (git add file name) only for one file 
3. git commit -m "your message (like project updated)"
4. git remote add origin Repository link
5. git push -u origin master (default branch is master)

*for project update cammad
1. git add . 
2. git commit -m "message"
3. git push
